Position

* position:static / relative / absolute / fixed / sticky * 좌표속성( left, right, top, bottom)

  • static - 기본값(반응형작업시 포지션 해제)
  • relative - 상대좌표(자기 자신을 기준으로 배치)
  • absolute - 절대좌표(부모요소를 기준으로 배치)
  • fixed - 고정좌표(뷰포트 기준으로 배치)
  • sticky - 상대,고정 혼합(스크롤 영역 기준으로 배치)

relative & absolute

* relative 대부분 absolute와 함께 사용. * 문서 상 원래 위치가 그대로 유지됨. * 위치를 이동하면서 다른 콘텐츠에 영향을 주지않음. * 원래 위치를 기준으로 상하좌우로 이동 가능.

+

absolute

* 문서상 원래 위치가 사라진다. * 위치를 이동하면서 다른 콘텐츠에 영향을 주지않음. * 부모 위치를 기준으로 상하좌우로 이동 가능. * 부모 크기에 영향을 주지 않음. * 페이지 스크롤 영향을 받음.

공지사항, 알림창...

fixed

* 문서상 원래 위치가 사라진다. * 위치를 이동하면서 다른 콘텐츠에 영향을 주지않음. * 부모 크기에 영향을 주지 않음. * 페이지 스크롤 영향을 받지 않음(고정). * 헤더, top버튼, 우측배너 * 3가지 스타일(filter, perspective, transform)

sticky

* 문서상 원래 위치는 그대로. * 다른 콘텐츠에 영향을 주지 않음. * 스크롤이 해당 콘텐츠에 닿으면 고정됨.
